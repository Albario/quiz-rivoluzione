## 🎯 Cos'è questo quiz

Il **Navigatore Biologico** è uno strumento gratuito di 2 minuti che aiuta l'utente a:

1. ✅ **Verificare la sicurezza** — un filtro iniziale per escludere chi non dovrebbe praticare il digiuno (gravidanza, DCA, diabete tipo 1, minorenni)
2. 🎯 **Identificare il proprio profilo** — tra 5 categorie: donna, metabolico, atleta, biohacker, libertà mentale
3. 📚 **Ricevere consigli personalizzati** — sui capitoli del libro più rilevanti per il proprio caso

Il quiz è pensato come **lead magnet** e strumento di pre-acquisto: aiuta l'utente a capire se il prodotto fa per lui prima di spendere soldi.
